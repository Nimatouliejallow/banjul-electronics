# Banjul Electronics - Style Guide

## Layout

The design uses a max-width container:

- Container max-width: 1000px
- Container padding: 0 20px

## Colors

### Primary

- **Primary (Blue)**: #0066CC
- **Accent (Orange)**: #FF6B35

### Neutral

- **Dark Navy**: #1A1A2E
- **Header/Navbar**: #0D0D1A
- **Text**: #333333
- **Secondary (Light Blue)**: #E8F4FC
- **Background**: #F5F5F5
- **White**: #FFFFFF
- **Border Gray**: #DDDDDD

## Typography

### Body Copy

- Font size: 16px
- Line height: 1.6

### Headings

- H1: 32px (36px in hero)
- H2: 24px
- H3: 20px

### Font

- Family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif
- Weights: 400 (normal), 700 (bold)

## Spacing

- Section margin-bottom: 40px
- Paragraph margin-bottom: 15px
- Container padding: 20px
- Header/Footer padding: 15px / 25px

## Components

### Buttons

- Background: #0066CC
- Text: #FFFFFF
- Padding: 12px 25px
- Border radius: 5px
- Hover background: #FF6B35

### Cards/Features

- Background: #FFFFFF                                                                                           
- Padding: 20px
- Border: 1px solid #DDDDDD
- Border radius: 8px

### Info Section

- Background: #E8F4FC
- Padding: 25px
- Border radius: 8px

### Forms

- Input padding: 12px
- Border: 2px solid #DDDDDD
- Border radius: 5px
- Focus border: #0066CC
